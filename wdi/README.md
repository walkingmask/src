wdi
======================
What day is ...?

Examples
------
```
$ wdi today
Thu
$ wdi 20000101
Sat
```

Options
------
```
yyyymmdd   The year, month, day
mmdd       The month, day of this year
-t, today  Today
```

Installing
------
```
cd wdi
bash install.sh
echo 'export PATH=$PATH:$HOME/bin' >>~/.zshrc
```

Uninstalling
------
```
cd wdi
sh uninstall.sh
```
