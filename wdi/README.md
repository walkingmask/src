# wdi

What day is ...?


## Examples

```
$ wdi today
Thu
$ wdi 20000101
Sat
```


# Options

```
yyyymmdd   The year, month, day
mmdd       The month, day of this year
-t, today  Today
```


## Install

```
cd wdi
bash install.sh
echo 'export PATH=$PATH:$HOME/bin' >>~/.zshrc
```


## Uninstall

```
cd wdi
sh uninstall.sh
```
