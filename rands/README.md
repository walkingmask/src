rands
======================
Output random string.

Examples
------
```
$ rands
2Y9b(5?I
$ rands
\4q_0&W2)N]vi\y7b8^c
$ rands -sS
25 MwK 2
```

Options
------
```
-l  length specifies the string length
-aAnsS
  a exclude small letters
  A exclude capital letters
  n exclude numbers
  s exclude symbols
  S include white spaces
-h  show help
```

Installing
------
```
cd rands
bash install.sh
echo 'export PATH=$PATH:$HOME/bin' >>~/.zshrc
```

Uninstalling
------
```
cd rands
sh uninstall.sh
```
